Projects

Website Style Guide

This website style guide describes various types of color schemes, fonts, and text styles. This was one of my favorite milestone projects in the Codecademy Front-end Development course. This project was used as a reference for flexbox, colors, font API's, and CSS hover and click properties.

Landscaping Makeover

After The Brother's Mancia landscaping company completed some great work in my yard, I decided to use pictures of my yard to give their website a new and improved look. I used css transitions and incorporated javascript to bring buttons to life and enhance user experience.

Tutoring Website

As a former math teacher, I tutored students outside of school and gained a following through word of mouth. I created my own tutoring website using HTML5 and CSS to mitigate scheduling conflicts. I incorporated a Calendly API so that my clients are able to choose times slots that are convenient for them.

Django Girls Blog

Using python, django, HTML, CSS, and git, I was able to create a working web application. I learned about Django ORM and Querysets, dynamic data in templates, and django admin/forms. I practiced using the command line and working in a virtual environment. It was very interesting learning how to run a local server and even deploy my blog.

Tea Cozy

In Codecademy's Front End Developer course, I designed this website after being given unique specifications. By completing this project, I was able to practice using a combination of border-box and flexbox, along with overlaying text and images. This project allowed me to use various methods of styling to determine the benefits of each.

Credit Card Validator

I wrote a program to optimize a credit card validation process. I based the program off of the Luhn alogirthm, which uses a check digit and multiple test credentials to validate identification numbers. I created two different versions (Version 1 and Version 2) to experiment with different methods.
